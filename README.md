# **IHome system - home automation control concept.**

**Centralized system**

All communications are carried out using one mini-server.
![](/gallery/ihome.png)

------------
**Decentralized system**

Communications are carried out using multiple mini-servers.
![](/gallery/ihome-cluster.png)

------------
- **[IHome KNX](https://github.com/xxandev/ihome-knx)** - microservice communication NATS-KNX/KNX-NATS
- **[IHome ModBus](https://github.com/xxandev/ihome-modbus)** - microservice communication NATS-MODBUS/MODBUS-NATS
- **[IHome HomeKit](https://github.com/xxandev/ihome-homekit)** - microservice communication NATS-HOMEKIT/HOMEKIT-NATS
- **[IHome Scenarist](https://github.com/xxandev/ihome-scenarist)** - microservice scenarist
